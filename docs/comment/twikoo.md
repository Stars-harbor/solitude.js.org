# Twikoo 评论系统

## 后端部署

[云函数部署](https://twikoo.js.org/backend.html)

> 推荐食用 `netlify` 部署

## 主题配置

```yaml
# Twikoo 配置信息
# Twikoo settings
twikoo: # https://twikoo.js.org/
  envId: your envid # 你部署twikoo的地址
  lang: zh-CN # 语言
  accessToken: # accessToken（在浏览器控制台cookies获取）
```
